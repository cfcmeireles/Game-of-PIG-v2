Game of PIG

There are three players. The player whose turn it is rolls the dice. The total of the roll is added to the current player's score, unless either dice comes up as a “one”. If this happens, this player gets no points for that turn and it is the next player’s turn. After each roll, the current player can either roll again, (assuming a “one” was not rolled) or if the current player feels that luck is running thin, they can pass to the next player. The first player to get to the agreed number of points or higher wins.

Oh, and if you roll two “ones” (snake eyes), your current score gets zeroed out. So don’t do that.
